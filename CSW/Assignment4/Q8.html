Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures content for the web and defines elements like headings, paragraphs, links, and images.

HTML Syntax

HTML is made up of elements enclosed in tags. Here's an example:

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is a paragraph.</p>
</body>
</html>
